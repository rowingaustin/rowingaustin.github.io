Responsive Design

In the last lesson, we built a web page to for the town of Franklin. We have already employed media queries, but let us work on our responsive design even further and add some additional features. In this assignment, we will add some JavaScript, part of which will help us build a hamburger menu for a responsive, small view main navigation experience.
